Process

How Skybridge works

The entire process takes just a few minutes. You stay in control at every step.

1

Connect accounts

Sign in to identify your job, protect your data, and resume later. Authenticate CloudAhoy and FlySto, then set your optional import filters.

2

Review the summary

Preview flights, hours, and missing registrations before any changes are made. Check data integrity and spot any issues.

3

Approve import

Start the import when you are ready, then download the results as a bundle. Watch progress in real-time.

Security

Built for private, review-first imports

Skybridge is designed to minimize risk: credentials are used only for the active job, and you always review before importing.

Credentials not stored

Timed retention
